R E A D I N G  P R E S E N T A T I O N :  L I V E

software studies: a lexicon, from interface to perl

Internationalization, or "i18n"

Universality VS. Otherness

As software become more social, the need for universality increases, and while companies are actively using resources such as Locales in order to provide global platforms, it seems to be inherently limited due to the constraints placed on technology by capitalism.

Exclusion becomes less acknowledged. Doing so would not be very profitable.

Interrupt

With the introduction of these interrupting elements, not only did it have a remarkable amount of self-regulating power, as it allowed roadstops to signal areas to stop a loop, it also brought interaction into the picture.

This kind of external agency seemed to be proof that things outside of this system existed.

When related to outside phenomena -- think of your body as a system, think of the traffic around you as a system, etc. Each has various markers and independently functioning elements that can interrupt one another.

  • "The interrupt can therefore be thought of, on an extended level, as the vector that not only constructs associations between actors, but also traverses varying operational spaces." (163)
  • "How do you recognize that you are speaking to a living person?" to which Derrida responded: "By the fact that they interrupt you." (162)

Lists

  1. Have always been used to organize and compile information.
  2. Has affected the way we store data and process information and memory.

Memory

Computer Memory VS. Human Memory, or human memory as seeming universally legitimate because of computer memory's proof of storage.

  • "Or, to put it a different way, the hypothesis is that computer memory is not just one possibly model of memory, it is the best model of memory." (185)

Obfuscated Code

Open source languages such as Perl are used to obfuscate code. This is a way for creativity to be infused into the rational, to purposely obscure and explore the pathways that lead to meaning.
